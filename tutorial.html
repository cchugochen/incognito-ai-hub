How to Get a Free Google Gemini API Key

An Easy Guide for "Incognito AI Hub" Users (2025 Edition)

如何申請免費的 Google Gemini API 金鑰

專為「無痕AI工作台 (Incognito AI Hub)」使用者設計的簡易教學 (2025年版)

Foreword

"Incognito AI Hub" provides powerful translation and recognition features by connecting to Google's Gemini AI model. To ensure your data security and privacy, the tool requires you to use your own API key. The following steps will guide you through the free application process.

前言

「無痕AI工作台 (Incognito AI Hub)」透過串接 Google 的 Gemini AI 模型來提供強大的翻譯與辨識功能。為了確保您的資料安全與隱私，本工具需要您使用自己的 API 金鑰。以下將引導您完成免費申請流程。

1

Go to Google AI Studio

First, click the button below to open the official Google AI Studio website. This is the platform Google provides for developers to test and obtain Gemini APIs.

前往 Google AI Studio

首先，請點擊下方按鈕，開啟 Google AI Studio 官方網站。這是 Google 提供給開發者測試與獲取 Gemini API 的平台。

Go to Google AI Studio
2

Get API Key

After logging into your Google account, click the "Get API key" button in the left-side menu. Then, on the main screen, click the blue "+ Create API key" button.

取得 API 金鑰

登入您的 Google 帳號後，點擊頁面左側選單的「Get API key」按鈕。接著，在主畫面點擊藍色的「+ Create API key」按鈕。

3

Create and Copy Your Key

In the pop-up window, the system will guide you to create a new Google Cloud project for this key (you usually just need to click agree). Once done, click "Create API key".

The screen will display a string of letters and numbers; this is your API key. Please click the copy icon next to it to copy the entire key.

建立並複製您的金鑰

在跳出的視窗中，系統會引導您為這個金鑰建立一個新的 Google Cloud 專案 (您通常只需點擊同意即可)。完成後，點擊「Create API key」。

畫面會顯示一組由許多英文字母和數字組成的字串，這就是您的 API 金鑰。請點擊旁邊的複製圖示，將它完整複製下來。

4

Paste into Extension Settings

Return to your Chrome browser. On the "⚙️ Settings" page of the "Incognito AI Hub" extension, paste the key you just copied into the "Gemini API Key" field and click "Save Settings".

Congratulations! You have now completed all the settings and can start using all the AI services of the extension.

貼到擴充功能設定中

回到 Chrome 瀏覽器，在「Incognito AI Hub」的「⚙️ 設定」頁面，將剛剛複製的金鑰貼到「Gemini API 金鑰」欄位中，並點擊「儲存設定」。

恭喜您！至此已完成所有設定，可以開始使用擴充功能的所有 AI 服務了。

Important Security Notice

Your API key is like a key to your house. Absolutely do not share it with anyone or post it on any public websites (like GitHub, forums, etc.) to prevent it from being misused and incurring unexpected charges.

重要安全性提醒

您的 API 金鑰就如同您家裡的鑰匙，**請絕對不要**將它分享給任何人，或張貼在任何公開的網站（如 GitHub、論壇）上，以免被盜用並產生非預期的費用。
