Incognito PrivoAI 隱私權政策 (Privacy Policy)

最後更新：2026年3月 / Last Updated: March 2026 (v28.0.2)

核心原則：您的資料，由您掌控

Our Core Principle: Your Data, Your Control

本擴充功能完全在您的瀏覽器本機端運作。我們沒有任何後端伺服器，也不會收集、儲存或看見任何您處理的內容或個人資訊。

This extension operates entirely locally on your browser. We do not have any backend servers, nor do we collect, store, or see any of the content you process or your personal information.

資料處理說明 / How Your Data Is Handled

1. 您的 Google Gemini API 金鑰 (Your API Key)

  • 您的金鑰會透過 Chrome 的標準儲存 API (chrome.storage.sync) 加密儲存在您自己的瀏覽器中，且只用於向 Google 的官方 API 端點發送請求。我們絕不會存取或傳輸您的金鑰。
  • Your key is stored encrypted within your own browser using Chrome's standard storage API (chrome.storage.sync). It is only used to send requests directly to Google's official API endpoint. We never access or transmit your key.

2. 您處理的內容：文字、圖片、語音 (Your Content)

  • 一般功能（翻譯、語音筆記）：內容資料僅會暫時儲存於您本機的 chrome.storage.local，並在功能使用完畢後立即清除，不會持久儲存。
  • General Features (Translation, Voice Notes): Content is temporarily stored in your local chrome.storage.local and is cleared immediately after use. Nothing is persisted.

  • 無痕對話（最高隱私）：所有對話（包含文字與圖片）僅存於您電腦的記憶體 (RAM) 中。關閉分頁後，紀錄將立即永久消失，無法復原，也不會寫入任何儲存空間。
  • Incognito Chat (Highest Privacy): All conversations (including text and images) are stored only in your computer's RAM. When the tab is closed, this history is permanently and irretrievably deleted — nothing is written to any storage.

3. （選用）本機 AI 模型 — 完全離線 (Optional: Local AI Model — Fully Offline)

  • 如果您啟用本機模型功能（Ollama 或 LM Studio），所有 AI 請求將發送至您設定的本機端點（例如 http://localhost:11434/v1），而非任何雲端服務。您的資料完全不會離開您的本機網路。
  • If you enable the local model feature (Ollama or LM Studio), all AI requests are sent to the local endpoint you configured (e.g., http://localhost:11434/v1), not to any cloud service. Your data never leaves your local network.

權限使用說明 / Permissions Usage

  • activeTab: 用於存取當前活動分頁的基本資訊（如分頁 ID），以觸發網頁翻譯功能。/ Used to access basic information about the active tab (such as tab ID) in order to trigger the webpage translation feature.
  • storage: 用於在本機安全地儲存您的 API 金鑰與所有設定（語言偏好、預設提示語等）。/ Used to securely store your API key and all settings (language preferences, preset prompts, etc.) locally.
  • debugger: 僅在您使用「翻譯當前網頁」功能時臨時啟用，用以擷取完整頁面截圖，擷取後立即分離，不會持續監控頁面。/ Temporarily enabled only when using the "Translate Current Page" feature to capture a full-page screenshot. It is detached immediately after capture and does not continuously monitor the page.

第三方服務 / Third-Party Services

  • 使用 Gemini 功能時，您的內容（文字、圖片、語音）將依據 Google 的隱私權政策傳送至 Google API 進行處理。
  • When using Gemini features, your content (text, images, voice) is sent to Google's API for processing, subject to Google's Privacy Policy.

  • 使用本機模型功能時，內容僅發送至您自行設定的本機端點，不涉及任何第三方服務。
  • When using the local model feature, content is only sent to the local endpoint you configured — no third-party services are involved.
