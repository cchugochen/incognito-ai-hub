Incognito AI Hub 隱私權政策 (Privacy Policy)

最後更新：2025年8月20日 / Last Updated: August 20, 2025

核心原則：您的資料，由您掌控

Our Core Principle: Your Data, Your Control

本擴充功能完全在您的瀏覽器本機端運作。我們**沒有**任何後端伺服器，也**不會**收集、儲存或看見任何您處理的內容或個人資訊。

This extension operates **entirely locally on your browser**. We do **not** have any backend servers, nor do we collect, store, or see any of the content you process or your personal information.

資料處理說明 / How Your Data Is Handled

1. 您的 Google Gemini API 金鑰 (Your API Key)

  • 您的金鑰會透過 Chrome 的標準儲存 API (chrome.storage.sync) **加密儲存在您自己的瀏覽器中**，且只用於向 Google 的官方 API 發送請求。我們絕不會存取您的金鑰。
  • Your key is stored **encrypted within your own browser** using Chrome's standard storage API (chrome.storage.sync). It is only used to send requests to Google's official API. We never access your key.

2. 您處理的內容：文字、圖片、語音 (Your Content)

  • 一般功能：在使用翻譯或辨識功能時，內容資料僅會**暫時**儲存於您本機的 chrome.storage.local，並在功能分頁關閉後清除。
  • General Features: When using translation or recognition, content is **temporarily** stored in your local chrome.storage.local and is cleared when the feature's tab is closed.

  • 無痕對話框 (最高隱私)：所有對話（包含文字與圖片）**僅存於您電腦的記憶體 (RAM) 中**。關閉分頁後，紀錄將**立即永久消失**，無法復原。
  • Incognito Chat (Highest Privacy): All conversations (including text and images) are **stored only in your computer's memory (RAM)**. When the tab is closed, this history is **permanently and irretrievably deleted.**

3. （選用）雲端日誌 (Optional Cloud Logging)

  • 此功能預設為關閉。只有在您明確於設定頁面填入**您自己的**端點網址時，才會將操作紀錄傳送到該網址。
  • This feature is disabled by default. Only when you explicitly provide **your own** endpoint URL in the settings will operational logs be sent there.

權限使用說明 / Permissions Usage

  • storage: 用於在本機安全地儲存您的 API 金鑰與設定。/ Used to securely store your API key and settings locally.
  • debugger: 僅在您使用「翻譯當前網頁」功能時臨時啟用以擷取完整頁面，擷取後立即分離。/ Temporarily enabled only when using the "Translate Current Page" feature to capture the full page, and it is detached immediately after.
